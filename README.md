Yosemite
=======

The initcfg Yosemite bootstrap configures the terminal to look like so: 

![](https://raw.githubusercontent.com/initcfg/yosemite/master/imgs/screenshot-no-command.png)

It is built on:
- zsh
- oh-my-zsh
- solarized
- Font Iconsolata

It also installs:
- google chrome
- homebrew
- iTerm2
- atom
